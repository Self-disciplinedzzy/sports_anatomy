摘要：本节课主要讲解了下肢动作的解剖学分析，重点围绕正脚背踢球和蹲起（深蹲）两个动作展开。通过环节受力分析法，将动作划分为不同阶段，分析各阶段下肢关节的运动形式、原动肌、工作条件及性质。正脚背踢球分为后摆和前摆阶段，涉及髋、膝、踝关节的协调运动，强调原动肌的向心工作与鞭打式发力；深蹲则分为下降和蹲起阶段，分别对应离心与向心工作，重点锻炼下肢伸肌群的力量与控制能力。课程旨在帮助学生掌握动作的解剖学机制，提升运动技术。

---

# 一、课程引入
介绍本节课内容为下肢动作的解剖学分析，主要讲解正脚背踢球和蹲起两个动作，下肢动作按运动模式分为蹬、摆、缓冲三类。

# 二、正脚背踢球动作分析
该动作为足球常见技术，使用脚背正面击球，具有力量大、准确性高的特点，包括助跑、支撑腿着地、击球腿后摆与前摆等环节。

# 三、正脚背踢球后摆阶段
髋关节伸、膝关节屈、踝关节伸，原动肌分别为臀大肌等髋伸肌、股二头肌等膝屈肌、小腿前群肌；工作条件为近固定，工作性质为向心工作。

# 四、正脚背踢球前摆阶段
髋关节屈、膝关节伸、踝关节屈，原动肌为髂腰肌等髋屈肌、股四头肌、小腿三头肌；工作条件为近固定，工作性质为向心工作。

# 五、正脚背踢球动作要点
后摆阶段需控制摆幅以获得最佳初长度，前摆阶段应大关节带动小关节，实现爆发式鞭打动作。

# 六、蹲起动作概述
蹲起又称深蹲，是复合性全身力量训练动作，用于增强腿部、臀部力量及核心稳定性，动作要求双脚与肩同宽，保持背部挺直，下蹲至大腿与地面平行。

# 七、深蹲下蹲阶段
髋关节屈、膝关节屈、踝关节伸，环节运动方向与外力相同且减速，原动肌位于对侧，包括臀大肌、股四头肌、小腿三头肌；工作条件为远固定，工作性质为离心工作。

# 八、深蹲蹲起阶段
髋关节伸、膝关节伸、踝关节屈，环节运动方向与外力相反，原动肌位于同侧，为臀大肌、股四头肌、小腿三头肌；工作条件为远固定，工作性质为向心工作。

# 九、深蹲训练意义
通过离心与向心工作的结合，有效发展下肢伸肌群的离心收缩能力，提高动作控制力与缓冲效果，增强力量与稳定性。

# 十、课程总结
强调掌握动作解剖学分析方法的重要性，能够准确识别原动肌及其工作特征，从而深入理解动作机制，提升运动技术水平。
