摘要：本节课主要讲解了运动动作的解剖学分析方法，重点介绍了动作分析的基本步骤和寻找原动机的环节受力分析法。通过负重深蹲等实例，系统阐述了如何描述动作起始姿势、划分动作阶段，并从环节名称与运动方向、关节名称与运动形式、环节运动方向与外力方向关系等方面进行解剖学分析，最终确定原动机及其工作条件和性质，帮助科学指导运动训练与健身实践。

---

# 一、动作分析的基本流程
动作分析包括描述开始姿势、划分动作阶段、分阶段进行解剖学分析以及最后的讨论与建议四个主要步骤。

# 二、开始姿势的描述
开始姿势需用简明语言描述身体状态，如负重深蹲的开始姿势为肩扛杠铃、身体直立、双脚与肩同宽站立。

# 三、动作阶段的划分
以负重深蹲为例，可划分为蹲下阶段和蹲起阶段两个基本阶段，便于后续分步分析。

# 四、动作分析的六个内容
包括环节名称和运动方向、关节名称和运动形式、环节运动方向与外力方向的关系、原动机名称、原动机工作条件、原动机工作性质。

# 五、环节与运动方向
环节指可活动的人体节段，如上肢、下肢、脊柱；运动方向根据动作实际确定，如投掷时上肢向前上，落地缓冲时下肢向下。

# 六、关节运动形式的确定
将环节运动转化为具体关节的运动形式，如落地缓冲时髋、膝关节屈，踝关节伸。

# 七、环节运动方向与外力方向的关系
主要考虑重力作用；当两者方向相反时，原动机位于运动方向同侧（反同）；方向相同且加速时，原动机在同侧（快同）；方向相同但减速时，原动机在对侧（慢反）。

# 八、原动机的判定方法
通过环节受力分析法判断原动机位置，结合“反同”“快同”“慢反”口诀记忆不同情况下的原动机分布规律。

# 九、原动机的工作条件
根据肌肉起止点固定情况判断工作条件，如近固定、远固定、上固定、下固定和无固定，反映肌肉收缩时的力学状态。

# 十、原动机的工作性质
分为向心工作（肌肉缩短）、离心工作（肌肉拉长）和静力性工作（长度不变），如仰卧起坐中腹直肌为向心工作，俯卧撑下降时肱三头肌为离心工作。
