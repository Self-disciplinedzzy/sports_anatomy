# 一、关节运动的基本概念
关节运动是指身体某一环节在特定平面内围绕某一运动轴进行的活动，其运动形式由关节面的形态决定，主要涉及运动轴的方向与数量。

# 二、屈和伸
屈和伸是发生在矢状面内绕冠状轴的运动，通常向前为屈，向后为伸，但膝关节及踝关节相反，小腿向后贴大腿为屈，足尖下垂为屈，钩足尖为伸。

# 三、内收和外展
内收和外展是发生在额状面内绕矢状轴的运动，环节向正中矢状面靠拢为内收，远离则为外展；手指和足趾以中指或第二趾为中轴规定收展方向。

# 四、回旋（旋转）
回旋是运动环节在水平面内绕垂直轴的运动，由前向内旋转为旋内（或旋前），由前向外旋转为旋外（或旋后），如前臂的旋前与旋后动作。

# 五、水平屈伸
水平屈伸是上肢或下肢在外展90度后，在水平面内绕垂直轴的运动，向前为水平屈，向后为水平伸。

# 六、环转
环转是运动环节近端固定、远端做圆周运动的连续动作，由屈、展、伸、收依次组合而成，轨迹呈圆锥形。
