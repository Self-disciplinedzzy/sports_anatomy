# 一、多关节肌的概念
- 多关节肌是指跨越两个或两个以上关节的肌肉，如肱二头肌、股四头肌、国生肌和前臂肌群
- 相对地，单关节肌只跨过一个关节，如三角肌、胸大肌

# 二、多关节肌的功能性特征
多关节肌因跨多个关节，在一个关节处充分发挥作用后，难以在另一关节充分发力或被拉长，表现为“顾此失彼”，具有主动不足和被动不足两种功能性特征

# 三、多关节肌的主动不足
主动不足指多关节肌作为原动机时，肌力在一个关节充分作用后，无法在其他关节继续充分发力，其实质是肌肉力量不足，常见于举重、体操等抓握动作中前臂肌群的表现。

# 四、多关节肌的被动不足
被动不足指多关节肌作为对抗肌时，在一个关节被拉长后，难以在其他关节进一步被拉长，其实质是肌肉伸展性不够，如跨栏时股后肌群在伸膝后影响伸髋的幅度

# 五、多关节肌原理在运动实践中的应用
1. 比赛中应避免多关节肌的主动不足和被动不足以保证最佳发力，如足球正脚背踢球时支撑腿位置的设计
2. 训练中可利用该原理增强肌肉力量与伸展性，如通过主动不足强化股四头肌力量，利用被动不足提高伸展性

# 六、思考题与小结
思考田径运动中存在多关节肌主动不足和被动不足的动作，并探讨在比赛和训练中如何应用相关原理；掌握多关节肌特性有助于优化技术动作，提升运动成绩
