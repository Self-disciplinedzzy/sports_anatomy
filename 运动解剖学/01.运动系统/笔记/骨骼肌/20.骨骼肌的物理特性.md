# 骨骼肌的物理特性概述
骨骼肌具有三种物理特性：伸展性、弹性与粘滞性，这些特性影响肌肉在运动中的表现与损伤风险

# 伸展性
伸展性指肌肉在外力作用下被拉长的特性，如体前屈时腘绳肌群被拉伸，良好的伸展性有助于提高关节灵活性和完成大范围动作

# 弹性
弹性指外力解除后肌肉恢复原长的能力，类似弹簧的形变与复原，弹性好则肌肉收缩回缩力强，动作效率更高

# 粘滞性
粘滞性是肌纤维内部、肌纤维之间及肌群间摩擦产生的阻力，影响肌肉快速收缩与舒张，受体温影响显著

# 温度对粘滞性的影响
体温升高时粘滞性降低，体温降低时粘滞性增高，因此热身活动可提升体温，减少粘滞性，预防运动损伤

# 冬季运动注意事项
冬季气温低，肌肉粘滞性高，需充分进行准备活动以提高肌肉温度，增强收缩速度，防止损伤

# 课堂总结与思考
总结骨骼肌的三大物理特性及其在运动中的应用，提出思考问题：骨骼肌的物理特性与骨骼的物理特性有何区别

**骨骼肌​**​

1. ​**​展性​**​：可被外力拉长（如运动时肌肉伸展）
2. ​**​弹性​**​：外力消失后恢复原状（如跳起后落地时肌肉回弹）
3. ​**​粘性​**​：收缩时因内部胶状物质产生阻力（影响快速动作效率）

​**​骨骼​**​

1. ​**​坚硬性​**​：由矿物质（如钙）和骨基质构成，硬度高，支撑身体结构
2. ​**​刚性​**​：不可主动变形，维持形态稳定（如保护内脏、形成关节）
3. ​**​无收缩性​**​：需依赖骨骼肌牵拉才能活动（如肌肉收缩带动骨骼运动）

​**​总结​**​：骨骼肌柔软、可主动收缩，通过展性和弹性完成动作；骨骼坚硬、不可收缩，提供稳定支撑。两者协同工作，实现人体运动
