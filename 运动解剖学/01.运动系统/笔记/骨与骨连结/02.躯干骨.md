摘要：成人的躯干骨由24块椎骨、1块骶骨、1块尾骨、1块胸骨和12对肋骨组成，共51块，参与构成脊柱、胸廓和骨盆。椎骨包括7块颈椎、12块胸椎和5块腰椎，具有椎体、椎弓、椎孔及7个突起的一般结构，并在不同部位呈现特殊形态特征。颈椎横突有横突孔，第一颈椎为寰椎，第二颈椎具齿突，第七颈椎棘突长且不分叉；胸椎椎体和横突上有肋凹，棘突斜向后下；腰椎椎体肥大，棘突宽短呈板状水平向后。骶骨由5块骶椎融合而成，呈倒三角形，内有骶管，上下分别连接腰椎和尾骨；尾骨为退化的3~4块尾椎融合而成。胸骨分为胸骨柄、胸骨体和剑突，胸骨角为重要体表标志。肋由肋骨和肋软骨构成，第1-7对为真肋，8-10对为假肋并形成肋弓，11-12对为浮肋。课程最后强调了各椎骨的鉴别要点，并布置了如何快速区分颈椎、胸椎和腰椎的课后思考题。

# 1. 躯干骨的组成与数目
> ![[02.躯干骨分类.png]]
成人的躯干骨包括24块椎骨（7块颈椎、12块胸椎、5块腰椎）、1块骶骨、1块尾骨、1块胸骨和12对肋骨，共51块，分别参与构成脊柱、胸廓和骨盆。

# 2. 椎骨的一般结构
> ![[02.椎骨.png]]
每块椎骨由前方的椎体和后方的椎弓构成，两者围成椎孔，所有椎孔相连形成椎管以容纳脊髓
椎弓根上下缘有椎上切迹和椎下切迹，相邻切迹形成椎间孔供脊神经通过椎弓发出7个突起：1个棘突、1对横突、1对上关节突和1对下关节突

# 3. 颈椎的结构特点
1. 颈椎共有7块，椎体小，棘突末端多分叉（第三至第六颈椎），横突上有横突孔为判断颈椎的重要标志
> ![[02.3到6颈椎.png]]
2. 第一颈椎称寰椎，无椎体、棘突和关节突
> ![[02.第一颈椎寰椎.png]]
3. 第二颈椎称枢椎，椎体上有齿突
> ![[02.第二颈椎枢椎.png]]
4. 第七颈椎称隆椎，棘突长且末端不分叉，可在体表触及
> ![[02.第七颈椎隆椎.png]]

# 4. 胸椎的结构特点
1. 椎体较大，椎体后方上下有上下肋凹，与肋骨头相关节
2. 横突末端前方有横突肋凹，与肋结节相关节
3. 棘突细长，斜向后下方，肋凹是识别胸椎的主要特征
> ![[02.胸椎.png]]

# 5. 腰椎的结构特点
椎体肥大，横断面呈肾形或扇形，椎孔呈三角形，棘突宽短，呈板状水平向后伸展，便于肌肉附着
> ![[02.腰椎.png]]
# 6. 骶骨的结构特点
1. 由5块骶椎融合而成，呈倒三角形，构成骨盆后壁
2. 前缘有凸起的岬，两侧有耳状面与髂骨形成骶髂关节
3. 内部为骶管，上通椎管，下端为骶管裂孔
4. 前面光滑有四对骶前孔，后面粗糙有四对骶后孔，正中有骶正中嵴
> ![[02.骶骨.png]]
# 7. 尾骨的结构特点
由3~4块退化的尾椎融合而成，呈三角形，上接骶骨，下端游离为尾骨尖
> ![[02.尾骨.png]]
# 8. 胸骨的结构特点
1. 位于胸前壁正中，分为胸骨柄、胸骨体和剑突三部分
2. 胸骨柄上缘有颈静脉切迹，两侧有锁切迹与锁骨形成胸锁关节
3. 胸骨柄与胸骨体交界处向前突出形成胸骨角，平对第二肋，是计数肋的重要体表标志
4. 剑突位于下端，扁而薄，末端游离
> ![[02.胸骨.png]]
# 9. 肋的结构特点
1. 共12对，由后部的肋骨和前端的肋软骨构成
2. 肋骨属扁骨，分前端、体部和后端
3. 后端膨大为肋头，有关节面与胸椎体肋凹相关节
4. 肋颈后外侧为肋结节，与胸椎横突肋凹相关节
5. 体部内面下缘有肋沟，供肋间血管神经通过
6. 第一至第七对肋直接连于胸骨为真肋，第八至第十对借肋软骨连于上位肋软骨为假肋，并形成肋弓，第十一至第十二对前端游离为浮肋
> ![[02.肋骨单根.png]]
> ![[02.胸廓整体观.png]]
