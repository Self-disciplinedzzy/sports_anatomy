# 一、体循环动脉概述
血液由左心室射出，经主动脉及其各级分支到达全身毛细血管，进行物质和气体交换后，通过静脉返回右心房，此过程称为体循环,且体循环动脉内流动的是富含氧气和营养物质的动脉血

# 二、动脉分布规律
动脉分为==器官外动脉==和==器官内动脉==
1. 器官外动脉分布具有与人体结构相适应、每一大局部有1~2条动脉干、分壁支和脏支、常与静脉神经伴行形成血管神经束、多位于身体屈侧或隐蔽部位、以最短距离到达分布器官、分布形式与器官形态相关、口径与器官大小及功能有关等特点
2. 器官内动脉分布则与器官构造有关

# 三、体循环动脉主干
体循环动脉主干为主动脉，分为==升主动脉、主动脉弓和降主动脉==
1. 降主动脉又分为胸主动脉和腹主动脉
2. 升主动脉起自左心室，发出左右冠状动脉
3. 主动脉弓向左后弯曲，发出头臂干、左颈总动脉和左锁骨下动脉
4. 胸主动脉延续主动脉弓，穿过膈肌移行为腹主动脉
5. 腹主动脉在第四腰椎平面分为左右髂总动脉

# 四、头颈部动脉
1. 头颈部动脉主干为颈总动脉，左颈总动脉发自主动脉弓，右颈总动脉起于头臂干
2. 颈总动脉上行至甲状软骨上缘分为颈内动脉和颈外动脉
3. 颈外动脉分支包括面动脉、颞浅动脉、上颌动脉等
4. 颈内动脉入颅分布于脑
5. 颈动脉窦为压力感受器，颈动脉小球为化学感受器

# 五、上肢动脉
1. 上肢动脉主干为锁骨下动脉，左起自主动脉弓，右起于头臂干，主要分支有椎动脉、甲状颈干、胸廓内动脉
2. 腋动脉为锁骨下动脉延续，分支分布于肩部和胸壁；肱动脉下行分为桡动脉和尺动脉，主要分支为肱深动脉；
3. 动脉可触脉搏，分支形成掌浅弓
4. 尺动脉参与形成掌浅弓和掌深弓

# 六、胸部动脉
- 胸部动脉主干为胸主动脉，分支有壁支（肋间后动脉、肋下动脉、膈上动脉）分布于胸壁、腹壁上部、脊髓等
- 脏支包括支气管支、食管支、心包支，分布于相应内脏

# 七、腹部动脉
![[04.腹主动脉.jpg]]
1. 腹部动脉主干为腹主动脉，分支分为壁支和脏支
2. 壁支分布于腹后壁和膈肌
3. 成对脏支包括肾上腺中动脉、肾动脉、睾丸动脉或卵巢动脉
4. 不成对脏支包括腹腔干、肠系膜上动脉、肠系膜下动脉，分别供应胃肝脾、小肠盲肠升结肠横结肠、降结肠乙状结肠直肠上部
![[04.腹主动脉肝胃.jpg]]
# 八、臀部与下肢动脉
![[04.盆腔动脉.jpg]]
1. 腹主动脉分叉为左右髂总动脉，再分为髂内动脉和髂外动脉
2. 髂内动脉为盆部动脉主干，分布于盆壁、臀部、股内侧及盆腔脏器；下肢动脉主干为髂外动脉延续而成的股动脉
3. 股动脉主要分支为股深动脉
![[04.下肢动脉大腿.jpg]]
4. 股动脉移行为腘动脉，分出胫前动脉和胫后动脉
5. 胫后动脉分布于小腿后部及足底
6. 胫前动脉延续为足背动脉，分布于足背并参与足底深支
