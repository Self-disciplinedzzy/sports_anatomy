摘要：本次课程介绍了心脏的位置、外形和结构。心脏位于胸腔纵隔内，大部分在身体正中左侧，前方对向胸骨和肋软骨，后方平对胸椎，与食管和主动脉相邻。外形呈倒置圆锥体，具有一尖、一底、两面、三缘和四沟。心脏分为四个腔室：左心房、左心室、右心房和右心室，左右心房间和心室间互不相通。心壁结构包括房室瓣（二尖瓣、三尖瓣）和动脉瓣（主动脉瓣、肺动脉瓣），保证血液单向流动。心脏不仅作为血液循环的动力泵，还具有内分泌功能，其结构特点对体育运动的科学指导具有重要意义。

# 一、心脏的位置
心脏位于胸腔纵隔内，约2/3在身体正中左侧，1/3在右侧，前方对向胸骨体及第2至第6肋软骨，后方平对第5至第8胸椎，上方连接大血管，下方紧贴膈肌，两侧邻近胸膜腔和肺。

# 二、心脏的外形
心脏外形似倒置前后略扁的圆锥体，具有一尖（心尖）、一底（心底）、两面（胸肋面和膈面）、三缘（下缘、右缘、左缘）和四沟（冠状沟、前室间沟、后室间沟、房间沟），分别标志心腔的分界

# 三、心脏的结构
心脏有四个腔室：==右心房、右心室、左心房、左心室==
- 由房间隔和室间隔分隔，左右心房和心室之间互不相通
- 心房与同侧心室通过房室口相通

# 四、心腔与血流通道
![[02.心脏瓣膜.png]]
1. 右心房接收上腔静脉、下腔静脉和冠状窦的血液，经右房室口流入右心室
2. 右心室通过肺动脉口将血泵入肺动脉
3. 左心房接收肺静脉含氧血，经左房室口流入左心室
4. 左心室通过主动脉口将血泵入主动脉

# 五、心脏瓣膜及其功能
1. 三尖瓣位于右房室口，二尖瓣位于左房室口，防止心室收缩时血液逆流至心房
2. 肺动脉瓣位于肺动脉口，主动脉瓣位于主动脉口，防止心室舒张时血液回流
3. 瓣膜开闭规律协调，保障血液单向流动

# 六、心脏的功能意义
心脏作为血液循环的动力泵，推动血液在全身循环，同时具有内分泌功能；掌握其形态结构对理解生理机制和指导体育运动具有重要意义
