# 一.生殖系统概述
生殖系统由内生殖器和外生殖器组成，主要功能为产生生殖细胞、繁殖后代和分泌性激素。内生殖器包括生殖腺、输送管道及附属腺，外生殖器显露于体表，体现性别差异并参与生殖过程。

# 二.男性生殖器
## 男性内生殖器
包括==睾丸、附睾、输精管、射精管、精囊腺和前列腺==
1. 睾丸为男性生殖腺，位于阴囊内，产生精子和雄激素
2. 附睾是精子成熟和储存的场所
3. 输精管为肌性管道，参与精索构成
4. 射精管由输精管与精囊腺导管汇合而成，开口于前列腺部尿道
5. 精囊腺分泌弱碱性液体构成精液主要成分
6. 前列腺包绕尿道起始段，有导管开口于尿道

## 男性外生殖器
包括阴茎和阴囊。阴囊容纳睾丸并调节其温度，阴茎为排尿和排精的共同通道

# 女性生殖器
## 女性内生殖器
包括卵巢、输卵管、子宫和阴道
1. 卵巢为女性生殖腺，产生卵细胞和女性激素
2. 输卵管输送卵细胞至子宫
3. 子宫为孕育胎儿的肌性器官，位于盆腔中央
4. 阴道为连接子宫与外阴的肌性管道，下端开口于外阴部

## 女性外生殖器
又称外阴部，包括阴阜、大阴唇、小阴唇、阴蒂和阴道前庭等结构

## 乳房
乳房为哺乳器官，位于胸大肌前方第三至第六肋间皮下组织中，由皮肤、乳腺和脂肪组织构成，与生殖系统在功能上密切相关
