# 一、呼吸系统的组成与功能
1. 呼吸系统由呼吸道和肺组成，具有呼吸、嗅觉和发音功能
2. 呼吸道包括鼻、咽、喉、气管和支气管，分为上呼吸道（鼻、咽、喉）和下呼吸道（喉以下部分），肺是气体交换的器官

# 二、鼻的结构与功能
鼻为呼吸道起始部，分为==外鼻==、==鼻腔==和==鼻旁窦==三部分
1. 外鼻呈锥体形，位于面部中央
2. 鼻腔被鼻中隔分为左右两腔，内衬粘膜，分为嗅部和呼吸部，具有加温、湿润空气及嗅觉功能
3. 鼻旁窦包括额窦、上颌窦、筛窦和蝶窦，参与调节空气温湿度并对发音起共鸣作用

# 三、喉的位置与结构
1. 喉位于颈前部，连接咽与气管，由甲状软骨、环状软骨、会厌软骨和勺状软骨构成支架
2. 甲状软骨前部形成喉结，会厌在吞咽时关闭喉口
3. 喉腔内有声带和假声带，声门裂为发声关键结构，喉既是呼吸通道也是发音器官

# 四、气管与支气管的结构特点
- 气管上接环状软骨，下端分叉为左、右支气管。右支气管粗短且走向较直，异物易落入
- 支气管进入肺后反复分支形成支气管树，结构为气管的延续

# 五、肺的位置与形态
肺位于胸腔内，膈肌上方，纵隔两侧，呈圆锥形，分为肺尖、肺底（膈面）、纵隔面和肋面。肺门为血管、神经和支气管进出部位。左肺分上下两叶，右肺分上、中、下三叶。

# 六、肺的结构与功能分区
1. 肺表面覆有胸膜，质软富有弹性，主要由支气管树、肺泡及毛细血管网组成
2. 导气部包括从叶支气管到终末细支气管的各级支气管，仅输送气体
3. 呼吸部包括呼吸性细支气管、肺泡管、肺泡囊和肺泡，是气体交换的主要场所，其中肺泡为核心结构。

# 七、肺的血液循环
肺有两套血管系统
- 一是肺动脉和肺静脉组成的肺循环，负责气体交换，为功能性血管
- 二是支气管动脉，来自体循环，为肺组织提供营养，属营养性血管

# 八、体育运动对呼吸系统的影响
适宜的体育运动可促进呼吸器官形态结构的改善，提高呼吸机能；不当运动则可能产生不良影响。掌握呼吸系统结构有助于科学开展体育锻炼
