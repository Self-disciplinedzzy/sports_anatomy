
摘要：本节课主要介绍了泌尿系统的组成与功能，包括肾、输尿管、膀胱和尿道的结构与生理作用。重点讲解了肾的位置、外形、内部构造及肾单位的组成，强调肾是生成尿液、调节水盐平衡、酸碱平衡和电解质平衡的重要器官。同时阐述了肾的血液循环路径、输尿管的分段与狭窄特点、膀胱的容量与形态变化，以及男女尿道的差异，并指出科学锻炼对泌尿系统的积极影响，剧烈运动可能引发运动性蛋白尿或血尿。

# 一、泌尿系统概述  
泌尿系统由肾、输尿管、膀胱和尿道组成，主要功能是生成尿液，排出代谢废物如尿素、尿酸、肌酐等，同时调节水盐、酸碱及电解质平衡，维持内环境稳定

# 二、肾的位置与外形
![[04.肾的解剖学结构.png]]
- 肾为成对实质性器官，形似蚕豆，位于脊柱两侧第十一胸椎至第三腰椎之间，右肾略低于左肾
- 内侧缘中部凹陷为==肾门==，是血管、神经、淋巴管和肾盂出入部位

# 三、肾的构造  
肾切面可见肾窦，内含==肾小盏、肾大盏、肾盂、血管、淋巴管、神经及脂肪组织==
肾实质分为肾皮质和肾髓质，皮质含肾小体和肾柱，髓质由15~20个肾锥体构成，顶端为肾乳头，有乳头孔开口于肾小盏

# 四、肾单位与尿液生成  
每个肾约有100多万个肾单位，为肾结构和功能基本单位，由肾小体（毛细血管球+肾小囊）和肾小管组成，参与滤过、重吸收和分泌过程

# 五、肾的血液循环  
1. 肾动脉起自主动脉腹部，分支为叶间动脉、弓形动脉、小叶间动脉，再分出球小动脉形成肾小球毛细血管网，出球后再次形成毛细血管网包绕肾小管，最后汇入肾静脉
2. 髓质区由直小血管供血

# 六、输尿管结构与分段  
- 输尿管为成对肌性管道，起自肾盂，经腹段、盆段和壁内段下行至膀胱
- 全长有三处狭窄，易发生结石嵌顿。

# 七、膀胱的特点  
膀胱为肌性囊状器官，成人容量350~500毫升，形状、大小和位置随尿量、年龄和性别变化，具有较大伸缩性

# 八、尿道的性别差异  
1. 男性尿道长16~20厘米，兼具排尿和排精功能，起自膀胱尿道内口，止于阴茎尿道外口
2. 女性尿道长3~4厘米，起自膀胱尿道内口，开口于阴道前庭，仅用于排尿

# 九、运动对泌尿系统的影响  
科学体育锻炼有益于泌尿系统健康，但剧烈运动可致肾血管收缩、缺血缺氧，增加肾小球滤过膜通透性，可能导致运动性蛋白尿或血尿
